JavaScript語言特性

Posted on 2019-08-23

JavaScript雖然和Java很類似，但卻沒有 Java 的靜態定型（static typing）及強型態確認（strong type checking）特性，JavaScript變數資料型態沒有宣告就可使用（動態定型，dynamic typing)，我們先前了解了變數範圍後，接下來深入的去理解JavaScript的特性。

閉包(Closure)

閉包的概念為一個函式操作到另一個函式作用範圍內的變數。但只有巢狀結構的函式才會形成閉包，如果內層函式引用到外層函式的區域變數，外層函式中的變數狀態就會被內層函式保留下來(形成封閉狀態，只有內層函式可存取，達到保護變數內容目的)。

範例2-1

var g;
function outer() {
    var a = 100;
    function inner() {
        console.log(a);
    }
    g = inner;
}
outer();
g();

outer是100
g是undefined

範例2-2

使用閉包方法

印出a
var name = "a";
var obj = {
    name: "b",
    getName: function () {
        return function () {
            return this.name;
        };
    }
};
var result = obj.getName();
console.log(result());

印出b的方法
var name = "a";
var obj = {
    name: "b",
    getName: function () {
        var that = this;
        return function () {
            return that.name;
        };
    }
};
var result = obj.getName();
console.log(result());

印出b的方法就是在宣告一個that在外面，裡面的that在使用外面宣告的

範例2-3

利用閉包記憶變數

function add() {
    var count = 0;
    return function () {
        return count += 1;  //keep住變數
    }
}
var user = add();
var custom = add();

當我們要顯示每一秒顯示0~10的數字，先用for迴圈然後使用setTimeout

for(var i = 0; i < 10; i++) {
    setTimeout(function() {
        console.log(i);
    }, i*1000);
}

咦！ 他印10個10耶 那就錯了

範例2-4

我們可以使用閉包解決印出重複地問題

for(var i = 0; i < 10; i++) {
    (function(i) {
        setTimeout(function() {
            console.log(i);
        }, i*1000);
    })(i)
}

Currying

Curring是利用閉包將傳入的變數保存下來，將多個參數的function，轉變接受一個參數的function，簡化傳入參數的一種方式。
好處是提高程式的彈性和可讀性,易於重複利用。

範例2-5

參數重複傳入

function cube(height, width, depth) {
    return height * width * depth;
}
console.log(cube(3, 2, 1));
console.log(cube(3, 2, 2));
console.log(cube(3, 2, 3));
console.log(cube(3, 2, 4));
console.log(cube(3, 2, 5));

範例2-6

將參數利用閉包保存

function cube(height) {
    return function (width) {
        return function (depth) {
            return height * width * depth;
        };
    };
}
console.log(cube(3)(2)(1));

Hoisting

hoisting是指「一種把宣告提升到其所區域內最頂端」的內部行為，但是提升行為僅只針對宣告的部分，對於變數的設定，並沒有跟著提升

範例2-7

將變數宣告提升

var x = 5;
(function () {
    console.log(x);
    var x = 10;
})();
console.log(x);

程式實際執行過程

var x = 5;
(function () {
    var x;
    console.log(x);
    x = 10;
})();
console.log(x);
x為undefined跟外面的5

範例2-8

將變數宣告提升

var a = 1;
function hoisting() {
    if (!a) {
        var a = 2;
    }
    console.log(a);
}
hoisting();

程式實際執行過程

var a = 1;
function hoisting() {
    var a;
    if (!a) {
        a = 2;
    }
    console.log(a);
}
hoisting();
